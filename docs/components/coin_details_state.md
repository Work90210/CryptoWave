/// Represents the state of the Coin Details screen.
///
/// This class encapsulates all the data and status information required to render
/// the coin details UI, including loading states, error messages, and the
/// actual cryptocurrency data.
///
/// It is used within the `CoinDetailsCubit` to manage and update the UI's state
/// as data is fetched or changes occur.
///
/// ### Props/Parameters:
///
/// *   `status`: The current status of the data fetching process. Defaults to `CoinDetailsStatus.initial`.
///     Possible values are:
///     *   `CoinDetailsStatus.initial`: The initial state before any data fetching begins.
///     *   `CoinDetailsStatus.loading`: Data is currently being fetched.
///     *   `CoinDetailsStatus.success`: Data has been fetched successfully.
///     *   `CoinDetailsStatus.failure`: An error occurred during data fetching.
/// *   `coin`: The detailed information about the cryptocurrency. Defaults to `null`.
/// *   `errorMessage`: A message describing any error that occurred. Defaults to `null`.
///
/// ### Usage Examples:
///
/// ```dart
/// // Initial state
/// final initialState = CoinDetailsState();
///
/// // Loading state
/// final loadingState = CoinDetailsState(status: CoinDetailsStatus.loading);
///
/// // Success state with coin data
/// final successState = CoinDetailsState(
///   status: CoinDetailsStatus.success,
///   coin: Coin(
///     id: 'bitcoin',
///     symbol: 'btc',
///     name: 'Bitcoin',
///     image: 'https://example.com/btc.png',
///     currentPrice: 50000.0,
///     priceChangePercentage24h: 2.5,
///   ),
/// );
///
/// // Failure state with an error message
/// final failureState = CoinDetailsState(
///   status: CoinDetailsStatus.failure,
///   errorMessage: 'Failed to load coin data.',
/// );
/// ```
